# RupLo 💸

**Rup**ee **Lo**g — *Log your money.*

A modern Android expense tracker built with Kotlin and Jetpack Compose. RupLo helps users record income and expenses, track their financial health, and manage spending — all with offline-first reliability and cloud backup.

---

## 📱 About

Many students and young professionals struggle to track where their money goes. RupLo solves this with a simple, fast, and reliable expense management system — built as a resume-quality project demonstrating modern Android development practices.
